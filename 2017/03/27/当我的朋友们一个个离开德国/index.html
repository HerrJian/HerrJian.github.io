当我的朋友们一个个离开德国

Veröffentlicht am 2017-03-27 | in C'est la vie | | Visitors

只是今天送走了一个好朋友Q，之前也已经回去了一个S，之后看样子也还是要回去一个Z。因为是同一个批次来的，虽然不是走项目或者走中介，但总是天南海北聚在一起就是缘分，将近两年的时间里大部分和朋友相处的时光里都有他们的陪伴，突然间要走掉三个，还是有点不习惯。

当我谈起朋友这两个字，心里总也很充实。朋友不需要天天黏在一起，也许有机会，大家就会相见，喝个酒，聊个天，撸个串儿，吹个牛逼，规划一下以后的美好生活，调侃一下过去的傻逼岁月，挺好的。只是当他们一个个的不能陪你喝酒聊天撸串儿吹牛的时候，更是因为当他们因为各种原因离开德国的时候，心里也总还是有点失落的，马上就开始设想以后会是怎样的一种相逢。

德国的生活，学习是有点苦逼的，在德语还不是很溜的时候，多少还是需要有些朋友来抒发内心的情绪的。两个星期除了和收银员说个Schoene tag noch再没多说过一句话的生活我过够了。曾经在语言班的一段时间和两个印度小哥玩儿的很开心，很会喝酒，也很会享受生活，和他们在一起也很开心，但现在想起来觉得那个时候的自己开心到没有了理想，不知道自己以后要做什么，真的是穷开心。后来陆续遇到了本文说到的三个陆续回去的好哥们儿，虽然来自天南海北，性格也迥异不同，但却合的来，从没有过不开心，只有开心难忘的美好回忆。

也不是说和德国人或者外国人交不了朋友，但始终是有一面文化差异的墙，那种开心是另一种开心，和国人在一起是那种自由自在你一言我一语玩笑都能懂的开心。说中国人报团的也不是没有道理，我做饭好吃不代表德国人或者其他外国人享受的了，我要吃火锅，我要吃麻辣牛肉，我要吃麻辣小龙虾的时候，很多外国人是没有这种口福的，因为他们吃不了辣，而我也不愿意为了迎合他们的口味去改变我的做菜风格，当然好朋友还是要区别对待。

当他们一个一个的离开，倒也没有上升到伤感的高度，就是觉得以后喝酒撸串儿吹牛逼没有这几个人了会觉得少了点什么。还是要做好自己，以后回国再聚啦。

相聚有时，后会无期，兄弟们好好发展！
